# Dockerized App Deployment on Cloud

This project demonstrates deploying a simple web application using Docker on a Linux cloud server.

## Technologies Used
- Docker
- Linux
- Nginx
- HTML

## Steps
1. Created a simple HTML web app
2. Built Docker image using Dockerfile
3. Ran container on cloud server
4. Exposed application on port 80

## Output
The app is accessible via the public IP of the cloud server.
